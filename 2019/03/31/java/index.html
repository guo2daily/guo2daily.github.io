3.3 数据类型

  • 3.3.1 整型

    • int 4字节 最常用 -2147483648~2147483647 超过20亿
    • short 2字节 -32768~32767
    • long 8字节-9223382036854775808~9223382036854775807
    • byte 1字节 -128~127
    • 长整型有后缀L，十六精致又前缀0x或0X，8进制又前缀0，最好不要使用八进制常数

  • 3.3.2 浮点类型

    • 用于表示小数部分的数值，可在浮点数值后面添加后缀D或d

      • float 4字节

        • 有后缀F或f，没有后缀F的浮点数值默认为double类型

      • double 8字节

        • 数值精度是float的两倍，也叫爽进度数值

  • 3.3.3 char类型

    • 原本表示单个字符，字面量值要用单引号括起来如‘A’，可表示为十六进制\u0000到\Uffff

    • 转移序列

      • 当心注释中的\u

  • 3.3.4 Unicode和char类型

    • 不要在程序中使用char类型

  • 3.3.5 boolean类型

    • 用来判断逻辑条件，整型值和布尔值不能相互转换

      • false
      • true

3.4 变量

  • 每个变量都有一个类型（type），大小写敏感

    • 3.4.1 变量初始化

      • 声明一个变量后，必须用赋值语句对变量进行显式初始化，声明可以放在代码中的任何地方

    • 3.4.2 常量

      • 利用关键字final指示常量。关键字final表示这个变量只能被赋值一次，一旦赋值后不能更改，习惯上final后的常量名使用全大写

      • 经常希望某个常量在一个类中的多个方法中使用，称为类常量，用static final设置一个类常量

        • 类常量的定义位于main方法外部，同一个类的其他方法中也可以使用
        • 如一个常量被声明为public，那么其他类的方法也可以使用这个常量

3.6 字符串

  • String greeting = “Hello”;

  • 3.6.1 子串

    • String类的substring方法可以从一个较大的字符串提取一个子串
    • String greeting = “Hello”；
      String s = greeting.substring(0, 3);
      创建了一个由字符“Hel”组成的字符串
    • 优点：易计算子串长度，s.substring(a, b)长度为b-a，子串“Hel”长度为3-0=3

  • 3.6.2 拼接

    • 使用+号链接两个字符串
    • 如需把多个字符串放一起，用一个界定符分隔，可使用静态join方法：
      String all = String.join(“ / “, “S”, “M”, “L”, “XL”);
      // all is the string “S / M / L /XL”

  • 3.6.3 不可变字符串

    • String没有提供修改字符串的方法，要修改用拼接方法

  • 3.6.4 检测字符串是否相等

    • s.equals(t)

  • 3.6.5 空串与Null串

    • 空串“”是长度为0的字符串
    • 是一个Java对象，有自己的串长度（0）和内容（空）
    • String可存放一个特殊值名null，表示没有任何对象与该变量关联
    • 检测字符串既不是null也不为空串
      if (str != null && str.length() != 0)
